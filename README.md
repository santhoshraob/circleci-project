# Node.js Hello World app - [![Build Status][status]][travis]

Assemble dependencies:

    npm install

Execute unit tests:

    npm test

Run the app (and then visit `http://localhost:5000/`):

    npm run-script run

Recommended to use [nvm](https://github.com/creationix/nvm).
